Development
===========

Preparation 
1.  npm install -g grunt-cli

Installation
1.  git clone https://git-wip-us.apache.org/repos/asf/olingo-odata4-js
1.  cd datajs
1.  npm install

Build datajs-x.x.x.js
1.  grunt build
*   Output is copied into the directory ...

Run demo
1.  grunt run

Run tests
1.  grunt test
